Event Capture

Thank you for taking part in the event capture. This is to capture the events that your browser sends to this page whenever you interact the blue-bordered box.

This information will help web sites make sure they are handling events correctly.

When you are ready, please click/tap the button that displays what input device(s) you have. Pointers include mice and pens.

Data Sent

This is all the data that has been sent. Each list of captured events is sent with your browsers userAgent string, which identifies the type of browser you have and a random string so that the events you send can be correlated together.
